VoxFlow

Ditado macOS feito para Portugues europeu: fala, recebe uma transcricao limpa, corrige uma vez e deixa a app reutilizar esse contexto nas proximas sessoes.

Qualidade primeiro, controlo sempre no teu Mac.

O setup recomendado usa OpenAI para transcricao e polish, mas as keys sao tuas. A landing nao tem backend, nao recolhe audio e nao guarda credenciais.

Como usar

Fluxo curto para instalar, configurar e ditar em PT-PT.

  1. 01

    Compilar a app

    Clona o repo, instala Xcode 15+ e executa `swift build` ou o Run do Codex.

  2. 02

    Adicionar a tua key

    Na app, abre Definicoes, escolhe OpenAI e cola a tua API key. A key fica no Keychain.

  3. 03

    Ditar com Option + Space

    Mantem premido, fala naturalmente em Portugues de Portugal e larga para finalizar.

  4. 04

    Corrigir e ensinar

    Edita o resultado, guarda a correccao e adiciona termos no vocabulario personalizado.

Modelos recomendados

O melhor equilibrio para uma app do genero WhisperFlow, mas afinada para PT-PT.

Etapa Modelo Motivo
Transcricao final gpt-4o-transcribe Maior qualidade para audio completo e prompts com glossario.
Preview live gpt-realtime-whisper Texto parcial enquanto falas, sem substituir a transcricao final.
Polish PT-PT gpt-5.5 Corrige pontuacao, termos e estilo sem inventar conteudo.
Poupar custo mini variants Baixa custo quando a qualidade maxima nao e critica.

Bring your own key

O site serve apenas documentacao. A app macOS recebe a tua chave localmente, guarda-a no Keychain e envia audio/texto diretamente para o fornecedor configurado.

01

Nenhuma API key fica no Vercel.

02

Sem proxy backend por defeito.

03

Historico e correccoes ficam em `~/.voxflow/`.

Adaptar ao teu vocabulario

Nomes de clientes, marcas, siglas e termos tecnicos devem ir para o vocabulario personalizado. As correccoes guardadas alimentam os prompts seguintes.

Adicionar glossario antes de gravacoes importantes.

Guardar correccoes quando mudares uma palavra recorrente.

Usar gpt-4o-transcribe para capturas finais e realtime apenas para preview.

Configurar em quatro passos.

Sem conta central, sem proxy escondido. Cada utilizador usa a sua própria key.

  1. 01

    Compilar

    Clona o repo e executa swift build, ou usa o Run do Codex.

  2. 02

    Adicionar key

    Escolhe OpenAI nas definições e cola a tua API key. Fica no Keychain.

  3. 03

    Ditar

    Prime Option + Space, fala em Português de Portugal e larga para finalizar.

  4. 04

    Ensinar

    Corrige o texto, guarda a correção e alimenta o vocabulário local.
